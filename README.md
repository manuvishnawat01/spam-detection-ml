# 📩 Spam Detection Web App

This is a simple machine learning web application that helps users identify whether a message is spam or not. The app takes a text message as input and uses a trained model to classify it instantly.

Currently, the project provides real-time spam detection with probability scores through a clean and minimal Streamlit interface. More features and improvements are being added to enhance accuracy and usability.

## 🌐 Live Demo

https://spam-detection-ml-dnterejyhhxvrpgb8ka5kr.streamlit.app/

## ✨ Features

* Enter a message
* Predicts Spam or Not Spam
* Shows prediction probability
* Simple Streamlit interface

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
